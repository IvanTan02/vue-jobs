# Vue Jobs
A simple Vue.js job portal built with JSON server.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
# Run Vue.js app
npm run dev
```

```sh
# Run JSON server
npm run server
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
